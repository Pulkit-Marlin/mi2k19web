Follow the given rules for images to load properly:

1. image name: 
	competition_category_name followed by 1 or 2 for left or right image respectively e.g.
	Music1.png 
	Music2.png
	Speaking Arts1.png

2. The competition_category_name is case-sensitive and should exactly match with that given in json file including spaces.

3. Image extension must be png

4. Image must be cropped properly so that only the figure to be displayed is visible with little or no extra space.
